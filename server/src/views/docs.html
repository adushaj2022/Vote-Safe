Vote Safe API Docs
/elections/{id} Delete election by id
/elections Returns all active elections
/user/me Requires a token, returns information about a user that is logged in
/election/user Returns elections created by a user
/sms/send-code Send verification code to user
/sms/verify-code Verify 6 digit code that the user provides
/user/register Creates new user
/user/login Allows a user to log in
/invite/accept Registering for an election
